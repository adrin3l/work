Plugin helps you redirect posts with 301 to homepage if they are expired

== Description ==

Plugin adds a new options in backend where you can choose an expiration date, so if the post expired will be redirected to homepage.

== Frequently Asked Questions ==

= What happends if i don't choose any expiration date ? =

If is not selected the expiration date it will not have any effect on front-end

== Installation ==

1. Upload automatic-posting to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
