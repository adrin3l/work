Plugin add tinymce to excerpt

== Description ==

Plugin adds posibility to use excerpt content with tinymce buttons.

== Frequently Asked Questions ==

= How i can use excerpt tinymce ? =

Tinymce for excerpt works exactly as the tinymce for content works.

== Installation ==

1. Upload automatic-posting to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
