Plugin helps you to convert one post in to page.

== Description ==

Plugin adds a new options in post edit where you can convert that post to page.


== Frequently Asked Questions ==

= What happends if i push to convert link? =

When convert link is pushed one ajax call is requested and post type is changed to page.

== Installation ==

1. Upload automatic-posting to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
