Plugin helps you to add realated posts to any post.

== Description ==

Plugin adds a new metabox in post edit where tou can choose if show or not related posts in front-end.



== Frequently Asked Questions ==

= How related posts works ? =

Related posts will get all articles from the same term taxonomies that the current post is, if none of taxonomy is choosen will default to 'category'.
