A simple and search engine friendly theme. Adspot ready with 6 widget-ready areas.


== Description == 

A search engine optimized website theme for WordPress. 6 widget-ready areas and a user friendly theme options page.

== Installation ==

1. Upload the whole plugin folder to your /wp-content/themes/ folder.
2. Activate the theme through the 'Themes' menu in WordPress.

OR

Just grab the StandardPack theme from the built in theme directory and choose "Install now", hit ok, then select the "Activate Theme" link.
